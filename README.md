# Data Analytics & Reporting Project

## Role: Data Analyst / Data Scientist

This project demonstrates end-to-end data analytics and reporting work including:
- Data extraction using SQL
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- KPI dashboarding using Power BI and Excel
- Basic statistical and machine learning modeling
- Audit-ready validation and reporting

## Responsibilities
- Built and maintained KPI dashboards using Power BI and Excel.
- Performed data extraction, cleaning, preprocessing, and validation using Python and SQL.
- Conducted exploratory data analysis to identify trends, anomalies, and operational gaps.
- Developed basic statistical and machine learning models for classification and prediction use cases.
- Ensured data accuracy, consistency, and audit readiness with validation procedures.

> Dataset and business context are anonymized for confidentiality.
